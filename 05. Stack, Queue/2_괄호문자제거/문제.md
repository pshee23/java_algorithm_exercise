# 문제

```#초급```&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;```#java```

- ## 설명
        
   입력된 문자열에서 소괄호 ( ) 사이에 존재하는 모든 문자를 제거하고 남은 문자만 출력하는 프로그램을 작성하세요.

<br/>
        
- ## 입력
        
    첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.
    
<br/>

- ## 출력
        
    남은 문자만 출력한다.

<br/>
        
- ## 예시

    ```java
    //입력
    (A(BC)D)EF(G(H)(IJ)K)LM(N)
    ```
    ```java
    //출력
    EFLM
    ```
