# 문제

```#초급```&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;```#java```

- ## 설명
        
   자연수 N이 입력되면 N!를 구하는 프로그램을 작성하세요.
   예를 들어 5! = 5 * 4 * 3 * 2 * 1 = 120 입니다.

<br/>
        
- ## 입력
        
    첫 번째 줄에 자연수 N(1<=N<=100)이 주어집니다.

<br/>

- ## 출력
        
    첫 번째 줄에 N팩토리얼 값을 출력하세요

<br/>
        
- ## 예시

    ```java
    //입력
    5
    ```
    ```java
    //출력
    120
    ```
