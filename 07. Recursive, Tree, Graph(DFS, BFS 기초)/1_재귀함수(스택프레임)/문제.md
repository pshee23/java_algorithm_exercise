# 문제

```#초급```&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;```#java```

- ## 설명
        
   자연수 N이 입력되면 재귀함수를 이용하여 1부터 N까지의 출력하는 프로그램을 작성하시오.

<br/>
        
- ## 입력
        
    첫 번째 줄은 정수 N(3<=N<=10)이 입력된다.

<br/>

- ## 출력
        
    첫째 줄에 출력한다.

<br/>
        
- ## 예시

    ```java
    //입력
    3
    ```
    ```java
    //출력
    1 2 3
    ```
