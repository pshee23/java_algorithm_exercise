# 문제

```#초급```&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;```#java```

- ## 설명
        
    알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는

    문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로 문자열을 압축하는 프로그램을 작성하시오.

    단 반복횟수가 1인 경우 생략합니다. 

<br/>
        
- ## 입력
        
    첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.
        
<br/>

- ## 출력
        
    첫 줄에 압축된 문자열을 출력한다.

<br/>
        
- ## 예시

    ### 예시 1

    ```java
    //입력
    KKHSSSSSSSE
    ```
    ```java
    //출력
    K2HS7E
    ```

    ### 예시 2

    ```java
    //입력
    KSTTTSEEKFKKKDJJGG
    ```
    ```java
    //출력
    KST3SE2KFK3DJ2G2
    ```
